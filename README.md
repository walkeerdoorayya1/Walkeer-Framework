🐍 Usage - Walkeer.py
The Walkeer.py script provides a menu-driven framework for scanning, payload building, and other penetration testing tasks.  

Run the script
`bash
python3 Walkeer.py
`

Menu Options
- [1] USB Port Scan → Scan connected USB devices  
- [4] Build Dropper → Generate Android reverse shell APK using msfvenom  
- [5] Run Payload → Launch Metasploit handler for reverse shell  
- [8] List Exploits → Search Android exploits in Metasploit  
- [9] Blue-Ducky → Scan Bluetooth devices  
- [10] OTG USB Run → Mount, copy, or unmount USB OTG devices

This script is intended for educational purposes only.  
Use it responsibly in controlled environments with proper authorization.
